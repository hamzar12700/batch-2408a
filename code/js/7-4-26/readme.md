# React JS Essential JS Revision

Ye repo students ke liye hai jo **React JS sikhne se pehle JavaScript ke essential topics** revise kar saken.

## Topics Covered
1. Variables & Data types
2. Functions (Arrow, Callback, Return)
3. Objects (Destructuring, Spread, Optional chaining)
4. Arrays (map, filter, find, slice, splice, push, pop, shift, unshift)
5. ES6 Features
6. Events & Conditional rendering


## How to Use

- Clone this repo
- Open `.js` files in any code editor
- Run using Node.js or in browser console
- Try modifying the examples to practice
